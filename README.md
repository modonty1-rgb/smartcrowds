# SMART CROWD Application

A modern, bilingual (Arabic/English) website for SMART CROWD (SMARTCROWD.SA) - a leading crowd management and event organization company in Saudi Arabia.

## Tech Stack

- Next.js 15+ (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- next-intl

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app router pages and layouts
- `components/` - React components
- `lib/` - Utility functions and configurations
- `messages/` - Internationalization content files

## Features

- Bilingual support (Arabic/English)
- RTL layout support
- SEO optimized
- High performance
- Admin dashboard
- Responsive design
